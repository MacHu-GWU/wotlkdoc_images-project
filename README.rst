Install
------------------------------------------------------------------------------

``wotlkdoc_images`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install wotlkdoc_images

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade wotlkdoc_images
